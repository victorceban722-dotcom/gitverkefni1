1. Hvað gera eftirfarandi Linux skipanir?
    * cd - Fer til fyrri möppu sem þú varst í.
    * ls - listar innihald möppu.
    * pwd - núverandi möppu.
    * mkdir - býr til nýja möppu með nafni.
2. Hvað gera eftirfarandi Git skipanir?
    * git clone - Afritar (clone) fjarkóða (remote repo) frá GitHub yfir á tölvuna þína.
    * git status - Sýnir núverandi stöðu, hvaða skrár hafa breyst, hvaða skrár eru staged eða unstaged.
    * git diff - Sýnir nákvæmar breytingar sem hafa verið gerðar í skrám síðan síðasta commit.
3. Hvað gera eftirfarandi Git skipanir saman? Hvaða gagn er að þeirri aðgerð?
    * git log - Sýnir sögu commits í repo. Hefur upplýsingar um hver gerði commit, tíma, og skilaboð.
    * git checkout - Skipar yfir á annan branch eða endurheimtir skráarbreytingar.
    * git branch - Sýnir alla branch-a í verkefninu og hvaða branch þú ert á núna.
4. Hvað er útgáfustýring (Version Control)? Kerfi sem skráir allar breytingar á kóða eða skjölum og Leyfir þér að fara til baka, sjá hver gerði breytingar, og vinna saman í hópi án þess að tapa vinnu.
5. Hverjir eru helstu kostir við að nota GIT? 
    * Skráir sögu breytinga (hver, hvenær, hvað).

    * Leyfir að vinna í mörgum branchum samtímis.

    * Auðvelt að vinna í hópi án árekstra.

    * Getur farið til baka ef eitthvað fer úrskeiðis.

    * Samhæfir verkefni á GitHub eða öðrum serverum.
6. Hversu oft telur þú að eigi að gera færslur (commit) í verkefni? 
    * Mjög reglulega

     * Þegar þú hefur lokið við smáhluta sem virkar.

     * Þegar þú bætir við nýrri virkni.

     * Þegar þú hefur lagað villu.

7. Hvað er átt við með “Working Directory”?” og “Staging Area” í GIT? 

    * Working Directory	Möppan á tölvunni þinni þar sem þú ert að vinna með skrárnar núna.
    * Staging Area	Svæði þar sem þú velur hvaða skrár fara í næsta commit (git add).
